# Wireframes

This directory contains SVG wireframe images for page designs.

## Using the create-page-wireframe skill

To create a new wireframe, use the `/create-page-wireframe` command in Claude Code:

```bash
/create-page-wireframe "Create a dashboard page with sidebar navigation, metrics cards, and a data table"
```

Or simply:

```bash
/create-page-wireframe
```

And Claude will ask you for the specification details.

## Example

See `example-login-wireframe.svg` for a sample wireframe of a login page.

## Wireframe Guidelines

- All wireframes are created as SVG files for scalability
- Standard desktop viewport: 1200x800
- Grayscale color scheme for clarity
- Labeled sections for easy reference
- Focus on structure and layout, not detailed design
